# DINO-ADVENTURE
Chromeの恐竜ゲームの再現です

## LEVEL SELECT

- [Easy](https://kenton116.github.io/DINO-ADVENTURE/src/easy/dino.html)
- [Hard](https://kenton116.github.io/DINO-ADVENTURE/src/hard/dino.html)

## HOW TO PLAY

敵（サボテンと鳥）をジャンプで避けて、ハイスコアを目指すゲームです。

スペースキーでジャンプができます。

Game Over!と表示されたら、Enterキーでリスタートできます。
